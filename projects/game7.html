AI Director Arena

Description:
A browser-playable Unity survival prototype featuring an AI Director that dynamically adjusts enemy pressure and healing opportunities based on real-time player performance.

My Role:
Gameplay Programmer, Systems Designer, UI Programmer

Tech:
Unity, C#, Web Build, Tailwind

Technical Highlights:
- Adaptive AI Director controlling pacing
- Enemy spawning system with dynamic difficulty
- Health pickups influenced by game state
- Kill counter, survival timer, and game over flow

Playable Build

Best experienced on desktop Chrome, Edge, or Firefox.
